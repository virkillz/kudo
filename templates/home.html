Build to be catalyst for Myanmar blockchain developer

There are many approaches to introduce blockchain and crypto ecosystem. We create Kudo on permissioned blockchain to simplify the security and create a controlled environment where people can start collaborate and see what they can do on the ecosystem.

Faster confirmation

Unlike 10 minutes bitcoin, Kudo confirmation time is 15 seconds.

Free transaction fee

Since this is built on the permissioned blockchain, we can set zero transaction fee and able to handle spam at the same time.

Bitcoin compatibility

We have backward compatibility with bitcoin, which means you can use existing libraries to mess around with.

Distribution

All token is pre-mined. We create and distribute 1 billion of token so people don't hesitate to give Kudo to other. This token has no value and so far have no intention to be listed on any market.
